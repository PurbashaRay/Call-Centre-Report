## Call Centre Report - Excel
This project is an Excel-based Call Centre Performance Dashboard designed to analyze and visualize the performance of call centre representatives over a financial year (FY). It provides insights into call trends, revenue generation, customer satisfaction, and individual agent rankings.
The dashboard helps managers and stakeholders make data-driven decisions to improve efficiency and customer service quality.

### 📁 Project Features<br>
📞Call Analysis
- Total number of calls handled by each representative<br>
- Weekly and yearly call trends<br>
- Peak call volume periods identification <br>

👨‍💼 Representative Performance<br>
- Calls handled per representative<br>
- Revenue generated by each representative<br>
- Overall performance comparison across agents<br>

⭐ Customer Feedback & Ratings<br>
- Average customer rating per representative<br>
- Rating distribution analysis<br>
- Identification of top-performing agents based on feedback<br>

🏆 Ranking System<br>
Automated ranking of representatives based on:
- Number of calls handled
- Revenue generated<br><br>

### 📊 Dashboard Highlights

- Interactive Excel dashboard with slicers and filters
- Visual charts for trend analysis (weekly, yearly)
- KPI cards for quick performance insights
- Comparative ranking table of all representatives<br><br>

### 🛠️ Tools & Technologies Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Visualization Tools
- Slicers & Filters
- Formulas (ROUND,XLOOKUP,RANK.AVG etc.)<br><br>
  
### 📈 Key Insights Generated

- Identification of top-performing call representatives
- Revenue contribution analysis per agent
- Customer satisfaction trends across FY
- Peak call load periods for workforce planning
- Performance gaps among representatives<br><br>
  
### 🎯 Objective
The main objective of this project is to:<br>

- Improve call centre operational efficiency
- Track individual representative performance
- Analyze customer satisfaction levels
- Support data-driven management decisions<br><br>
